### 欢迎使用GalIsland——galgame管理工具
✨ 功能特点

🎮 游戏库管理 - 添加、编辑、删除游戏，支持多种排序和筛选方式
🔍 VNDB 集成 - 自动从 VNDB 获取游戏信息、封面、截图、评分等
🏷️ 智能标签 - 从 ggbases 自动获取中文标签
🌐 AI 翻译 - 使用 DeepSeek AI 自动翻译游戏简介
🖼️ 图片管理 - 本地存储游戏封面和截图，支持拖拽上传
📁 分类管理 - 自定义分类，灵活组织游戏库

**新版更新**
    转区运行，基于LE转区实现，感谢LE开源社区。
    自动获取存档地址，一键直达。

**特殊说明**
如抓取不到信息，或抓取错误信息，可手动调整。从 ggbases 获取中文标签需要科学上网（v1.1.0可直接获取)。游戏封面与截图会保存到resources/data路径下，下载速度取决于网速，更新版本后请把旧版resources/data/categories.json，games.json和vndb_cache.json。以及resources/games文件夹替换到新版同目录文件。
